# Enemy Battle Game

A turn-based battle game built in C++ where the player fights against enemies that can attack and duplicate themselves over time.

## Features

* Object-oriented Enemy class design
* Constructors and copy constructors
* Enemy duplication system
* Randomized attack and damage mechanics
* Dynamic enemy management using vectors
* Turn-based gameplay loop
* Health tracking and enemy removal system

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Classes and Member Functions
* Vectors
* Algorithms
* Random Number Generation

## What I Learned

This project helped me strengthen my understanding of object-oriented programming, copy constructors, static variables, vectors, and game logic design in C++.
